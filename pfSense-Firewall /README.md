# pfSense Firewall

## Overview

This project demonstrates firewall administration and network segmentation using pfSense.

## Skills Demonstrated

- Firewall configuration
- VLAN implementation
- Network segmentation
- DNS configuration
- Network troubleshooting

## Technologies Used

- pfSense
- VLANs
- DNS
- Proxmox VE

## Outcome

Configured a firewall environment, implemented VLAN segmentation, and secured systems behind managed firewall rules.
