# Apache PHP Web Server

## Overview

This project demonstrates the deployment and configuration of a web server environment using Apache and PHP within a Linux container.

## Skills Demonstrated

- Web server deployment
- Apache configuration
- PHP installation and testing
- Linux system administration
- Web application hosting

## Technologies Used

- Linux
- Apache
- PHP
- Proxmox VE

## Outcome

Successfully configured a Linux-based web server environment and verified functionality through hosted test pages and web application access.
