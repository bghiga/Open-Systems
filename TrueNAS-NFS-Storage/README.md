# TrueNAS NFS Storage

## Overview

This project demonstrates the configuration and integration of network-attached storage using TrueNAS and NFS.

## Skills Demonstrated

- Network storage administration
- NFS share configuration
- Storage management
- Permissions management
- System troubleshooting

## Technologies Used

- TrueNAS
- NFS
- Linux
- Proxmox VE

## Outcome

Created and mounted shared storage resources while troubleshooting connectivity and permission-related issues.
