# MIS 4397 - Open Systems

## Overview

This repository contains projects and labs completed as part of MIS 4397 (Open Systems) at the University of Houston.

The coursework focused on virtualization, Linux administration, networking, storage management, web hosting, firewall configuration, and system recovery using enterprise technologies.

## Technologies Used

- Proxmox VE
- Linux Containers (LXC)
- Virtual Machines
- Apache
- PHP
- Virtualmin
- TrueNAS
- NFS
- pfSense
- VLANs
- DNS
- Linux Command Line

## Skills Demonstrated

- Virtualization and container management
- Backup and disaster recovery
- Snapshot creation and rollback
- Linux system administration
- Web server deployment
- Domain hosting and DNS configuration
- Network storage management
- Firewall and VLAN configuration
- Troubleshooting and system recovery

## Projects

### 1. Proxmox Virtualization & Backup Management

Configured and managed virtual machines and Linux containers in Proxmox VE. Performed backups, snapshots, restores, and rollback testing.

### 2. Linux Administration

Managed Linux containers and practiced user privilege management using Linux command-line tools.

### 3. Apache/PHP Web Server Deployment

Installed and configured Apache and PHP on a Linux container and hosted test web applications.

### 4. Virtualmin Domain Hosting

Configured Virtualmin and hosted multiple domains while implementing local DNS resolution.

### 5. TrueNAS Network Storage

Created and mounted NFS shares using TrueNAS and integrated network storage with virtual machines.

### 6. pfSense Firewall & VLAN Configuration

Configured a pfSense firewall, implemented VLAN segmentation, and secured systems behind a managed firewall.
