# Virtualmin Domain Hosting

## Overview

This project demonstrates the configuration and management of domains using Virtualmin within a Linux hosting environment.

## Skills Demonstrated

- Domain management
- Virtualmin administration
- DNS concepts
- Web hosting configuration
- Linux administration

## Technologies Used

- Virtualmin
- Linux
- DNS
- Proxmox VE

## Outcome

Configured hosted domains and implemented local DNS resolution to support web application access and administration.
